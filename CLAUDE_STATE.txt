## Key Technical Details

### BSFixedString
- Defined as: `typedef StringCache::Ref BSFixedString;` in GameTypes.h
- StringCache::Ref has member: `const char* data`
- Identical strings share same pointer (string interning)
- Hash function hashes pointer address for O(1) lookup

### Version Numbers
- SE: SKSE 2.0.20 + Runtime 1.5.97 (0x01050610)
- AE: SKSE 2.2.6 + Runtime 1.6.1170 (0x01064920)
- Version controls which memory addresses are used - MUST match game version!
